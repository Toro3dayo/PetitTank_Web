# ぷちタンク

ブラウザ上で遊べるトップダウン視点の戦車ゲームです。WASD で移動、Space で砲撃して敵を全滅させましょう。スマートフォンでは画面左下のパッドと右下の砲弾ボタンが使えます。

## 遊び方
- Space または砲弾ボタンでゲーム開始。
- WASD で移動方向を決め、砲弾を壁にバウンドさせながら敵を倒します。
- Wave は 3 段階。全て撃破すると勝利です。
- 被弾するとゲームオーバー。R キーでいつでもリスタートできます。

## ファイル構成
- `index.html`：Canvas とタッチ UI の配置。
- `style.css`：レイアウトとタッチ UI のスタイル。
- `js/`：ゲームロジック（入力・モデル・描画・制御）のモジュール群。
